# LimitlessStudio

Static landing page for Limitless Studio, built for video editing, videography, photography, music videos, real estate visuals, promotional ads, effects, animations, and transitions.

## Project Files

- `index.html` - landing page markup
- `style.css` - core responsive page styling
- `luxury.css` - luxury visual polish layer
- `script.js` - mobile navigation, reveal effects, and autoplay pause/resume
- `assets/` - local images, icons, and video media

## Local Preview

Open `index.html` directly in a browser, or use a simple static server from this folder.

## Publish With GitHub Pages

1. Create a new GitHub repository.
2. Upload everything inside this folder into the repository root.
3. Go to `Settings > Pages`.
4. Set source to `Deploy from a branch`.
5. Select `main` and `/ (root)`.
6. Save.

## Notes

- This page is self-contained.
- Video files are included locally so the page can run as a static website.
- `.nojekyll` is included for GitHub Pages compatibility.
